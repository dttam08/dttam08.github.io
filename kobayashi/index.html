Feature

三田市の歯医者「小林デンタルクリニック」は、かかりつけ歯科医になれるよう、さまざまな取り組みを実施しております。
温かみのある、より良い歯科医療を提供し、この歯医者にまた来たい、この先生に治療を任せたい、
と感じていただけるような歯科医院を目指します。

家族で通いやすい
ファミリーサポートが充実

院内はバリアフリー設計となっており、足腰の不自由な方やベビーカー連れの方もストレスなく移動することが可能です。完全個室制ですので、プライバシーが保たれ、安心して歯科治療に専念できます。

Feature 01

拡大視野での精密治療

当院では、常にマイクロスコープや歯科用ルーペを用いて視野を拡大し、治療の精度を可能な限り高めております。精密治療は、歯を残すだけでなく、審美性や耐久性を向上させる上でも欠かすことができません。

Feature 02

徹底した衛生管理で
安全な診療環境を維持

診療に使用した器具は、必ず滅菌処理を施し、個別にパッキングしております。マスクや手袋、患者さまが使用するエプロンなどは、使い捨て品を採用し、ヘッドレストのカバーも患者さまごとに交換しております。

  • 個別の滅菌パック

  • ヘッドカバーは患者様毎に交換

Feature 03

  • Feature 04

    分かりやすい説明と
    丁寧なカウンセリング

    歯科治療は、とても専門性の高いものなので、できる限りわかりやすい言葉でご説明します。当院では、治療内容が簡潔に記された説明カードをお渡することで、理解をより深めていただけるよう努めております。

  • Feature 05

    心のこもった医療サービス

    患者さまは、痛みや不安を抱えた状態で歯科医院を訪れます。人によっては、情緒が不安定になっていることも珍しくありません。そうした心境を理解した上で、少しでも気持ちを楽にしていただけるような接遇を心がけております。どなたでも安心して通える歯科医院を目指します。

Feature 06

痛みが少なく、そして
怖くない歯科治療

歯科治療に伴う痛みや不安、恐怖心などはできる限り軽減されるべきものです。そこで当院では、局所麻酔を施す際にも痛みが和らぐよう、さまざまな工夫を凝らしています。外科手術を伴うインプラント治療においては、静脈内鎮静法を行っております。そうすることで、いたくない、怖くない歯科治療が可能となります。

  • 表面麻酔

  • カートリッジウォーマー

  • 電動麻酔器

小林デンタルクリニックは、この地域のかかりつけ歯科医になれるよう、さまざまな取り組みを実施しております。何よりまず大切なのは、患者さまが安心して治療を受けられる環境です。とくに感染症対策には力を入れており、器具の滅菌はもちろんのこと、それぞれの診療室は完全な個室となっています。

治療から次回のアポイントまでを個室で完結させることが可能です。歯科治療はとてもデリケートなものなので、プライバシーを守ることも医療人の役目と考えております。個室であれば、プライバシーもきちんと守られます。また、院内はバリアフリーとなっており、小さなお子さまからお年寄りまで、どなたでも不自由なく移動できる環境が整っています。

温かみのある、より良い歯科医療を提供し、この歯医者にまた来たい、この先生に治療を任せたい、と感じていただけるような歯科医院を目指します。お口のことで何かお悩みがあれば、どんな些細なことでも構いませんので、お気軽に当院までご相談ください。
