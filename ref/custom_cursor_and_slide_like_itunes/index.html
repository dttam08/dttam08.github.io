Solution

More Solution +

Dryers

「環境改善」と「コスト削減」を同時に叶える、西村鐵工所のオリジナル。
「世の中に存在しない、でも必要なもの」を探求した製品。

  • CDドライヤー

    特許製品・ディスク型伝導加熱式液体乾燥機です。円盤型の液体乾燥装置としては多くの支持をいただいている製品です

    More

  • A-VCDドライヤー

    真空環境下でも安定した乾燥・濃縮が可能で、有機溶剤の回収や沸点を下げて低温乾燥が可能になりました

    More

  • E-CDドライヤー

    電気加熱式で化石燃料を燃やさないため、工場からのCO2排出削減に貢献します

    More

  • H-VCDドライヤー

    ヒートポンプで圧縮し、加熱源として再利用することで、エネルギー消費を数分の一まで減らします

    More

Conveyors

国内特許、海外商標、輸出実績もあり、
国内外から高い評価をいただいています。

  • インターナルバケットコンベアI-Bコンベヤ

    バケットを内側にすることによって、スムーズな積み込み、無理のない自然排出など、数々の特長・メリットを持った、きわめてメンテナンス頻度が少ない画期的なコンベアです

  • 水平・垂直自在型バケットエレベーター NZコンベヤ

    I-Bコンベアで培ったノウハウを駆使した、メンテナンス頻度が少ない水平垂直軌道のバケットエレベーターです
